# WebSocket 채팅 애플리케이션

Spring Boot와 WebSocket을 사용한 실시간 채팅 애플리케이션입니다.

## 주요 기능

- 실시간 양방향 통신 (WebSocket)
- 채팅 메시지 데이터베이스 저장 (MySQL)
- 채팅 히스토리 자동 로딩
- 실시간 메시지 브로드캐스트

## 기술 스택

- Spring Boot 3.2.5
- WebSocket
- Spring Data JPA
- MySQL
- Lombok

## 실행 방법

```bash
./gradlew bootRun
```

애플리케이션은 http://localhost:8080 에서 실행됩니다.
